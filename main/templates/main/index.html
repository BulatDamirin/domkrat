Расходные материалы

Назначение и применение

Цанги

Цанговый зажим состоит из корпуса и клиньев из прочной высоколегированной стали. Служит для сохранения напряжения в арматурном элементе, созданного гидродомкратом.

Устанавливается на обоих концах арматуры.
При натяжении материала, цанги закрываются, надежно фиксируя удлинение арматуры в пределах металлоформы

Ресурс цанги - не менее 500 циклов натяжения для арматуры класса Ат400С, Ат600, Ат800.

Захваты

Цанговый захват состоит из 3-х стальных лепестков, соединенных пружинами. Служит для надежного захвата арматурного элемента.

Установлен в корпусе гидроцилиндра. При движении штока захват закрывается, удерживая арматурный элемент.

Ресурс захвата - не менее 500 циклов натяжения для арматуры класса Ат400С, Ат600, Ат800.

Кассеты

Кассета состоит из корпуса, захвата, пружины и упорной гайки.
Служит для быстрой и удобной смены узла захвата, при переходе на другой диаметр арматурного элемента.

! Одна кассета предусмотрена в базовой комплектации гидродомкрата.
При необходимости кассеты для разных диаметров арматуры приобретаются отдельно.

Ресурс корпуса кассеты - не ограничен.
